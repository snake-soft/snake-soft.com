Was ist Python

Python ist eine höhere Programmiersprache, welche die objektorientierte, aspektorientierte und funktionale Programmierung unterstützt.

Die Sprache selbst ist seit Version 3.0 vollständig objektorientiert.

Sie ist sehr vielseitig und hat einige sehr nützliche Eigenschaften, welche nicht nur dem Entwickler zugute kommen.

Doch zunächst sei erwähnt, dass Python aufgrund seiner Flexibilität sehr standardisiert wurde.

Ein Aspekt dieser Standardisierung ist der Zen of Python (deutsche Übersetzung in klammer):

Beautiful is better than ugly.

Explicit is better than implicit.
